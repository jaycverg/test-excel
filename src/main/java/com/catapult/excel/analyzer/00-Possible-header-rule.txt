A possible header:
1. Must have at least one adjacent cell to the right or going down
2. Has a bold text
3. Has a background
4. Horizontal header
   a. if all the cells to right are adjacent
   b. if the adjacent cells to right are similar in style with the first cell to the left
   c. a second or more rows might be a header if the cells style are similar with their top cells
      and the number of cell count to right are similar to the cell count of the previous row header
5. Vertical header
   a. if all the cells going down are adjacent
   b. if the adjacent cells going down are similar in style with the first cell at the top
   c. a second or more columns might be a header if the cells style are similar with their left cells
      and the number of cell count going down are similar to the cell count of the previous column header
